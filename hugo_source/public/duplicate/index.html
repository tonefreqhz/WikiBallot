The most important page on this device

This device is a sovereign node in the Great Withholding network. Every copy you make is another node. No centre. No server. No platform that can deplatform it.

Give it to someone REAL. Face to face where possible. The network grows person to person. Not algorithm to algorithm.

What you need

  • A USB stick, 16GB minimum — £3–5
  • Any computer
  • 10 minutes

Steps — Windows

  1. Plug in a blank USB stick
  2. Open File Explorer
  3. Select all contents of this dongle (Ctrl+A)
  4. Copy (Ctrl+C)
  5. Navigate to the new USB stick
  6. Paste (Ctrl+V)
  7. Wait for copy to complete
  8. Eject safely

Steps — Mac

  1. Plug in a blank USB stick
  2. Open Terminal
  3. Run: cp -r /Volumes/DOUGHFORGE/* /Volumes/[NEW_STICK]/
  4. Eject: diskutil eject /Volumes/[NEW_STICK]

Steps — Linux

  1. Plug in a blank USB stick
  2. Run: rsync -av /media/[user]/DOUGHFORGE/ /media/[user]/[NEW]/
  3. Sync: sync

Seed to IPFS (makes it permanent)

Run tools/ipfs_seed.sh from the dongle.

No server required. No hosting fees. No takedown possible. The content lives as long as one person seeds it.
